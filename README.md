## 🌴 The Wild Oasis - Cabin Management System

The Wild Oasis is an advanced internal hotel cabin management system designed to streamline the day-to-day operations of a hotel. Built with **React** and powered by cutting-edge state management libraries, it provides an intuitive and visually appealing experience for internal employees to manage bookings, cabins, users, and settings.

🔗 **Live Demo:** [The Wild Oasis](https://the-wild-oasis-beta2.vercel.app/)

---

### 🚀 Features

#### 1. **Advanced State Management**
- Utilizes **React Query**, **Context API**, and **Redux** for efficient and dynamic state handling.
- Handles complex data interactions and API calls with minimal performance overhead.

#### 2. **Interactive Dashboard**
- Visual summary of daily check-in activities.
- **Area Chart** for visualizing daily sales trends.
- **Pie Chart** summarizing guest duration statistics.
- Real-time updates and easy-to-read data presentation using **Recharts**.

#### 3. **Cabin Management**
- Displays a comprehensive list of cabins with:
  - Price
  - Maximum Guest Capacity
  - Discount Availability
- Context menu for **Duplicating**, **Editing**, and **Deleting** cabins.
- Advanced **Filtering** (by discount) and **Sorting** (by name, price, capacity).

#### 4. **Booking Management**
- Displays all bookings with pagination and includes the following details:
  - Cabin Name
  - Guest Name
  - Booking Dates
  - Booking Status (Unconfirmed, Checked-In, Checked-Out)
  - Total Price
- Context menu for **Viewing Details**, **Deleting**, **Check-In**, and **Check-Out**.
- Advanced **Filtering** (by status) and **Sorting** (by booking date and total price).

#### 5. **User Management**
- Employee registration and profile management.
- Authentication system with **Supabase** as the database.
- Profile updates including picture and password change.
- **Logout** functionality for a secure session end.

#### 6. **Settings Management**
- Authenticated users can update essential cabin settings such as:
  - Minimum and Maximum Booking Nights
  - Maximum Capacity
  - Breakfast Price

#### 7. **User Experience**
- Supports both **Dark and Light Modes**.
- Seamless login and signup via **Email**.
- Responsive and interactive design for a smooth user experience.

---

### 🛠️ Tech Stack

- **Frontend:** React, React Query, Redux, Context API
- **Charts:** Recharts
- **Database:** Supabase
- **Authentication:** Supabase Auth
- **State Management:** React Query, Redux, Context API
- **Deployment:** Vercel

---

### 🔒 Authentication

- The app uses **Supabase Auth** for secure user authentication and profile management.
- Users can update their profile information, including their profile picture and password.

---

### 🌗 Dark/Light Mode

- Toggle between dark and light modes effortlessly from the settings menu, enhancing usability and personal preference.
